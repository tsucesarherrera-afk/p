Guía Rápida de Solución

1

Verifica credenciales: URL y Anon Key de tu proyecto Supabase (Project Settings → API).

2

Confirma el email: En Supabase Dashboard → Authentication → Users, busca tu usuario y haz click en "Confirm email".

3

Verifica el trigger: En SQL Editor ejecuta: SELECT * FROM pg_trigger WHERE tgname = 'on_auth_user_created';

4

Verifica el perfil: Ejecuta: SELECT * FROM public.profiles;
